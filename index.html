WhatsUp

Bienvenido a WhatsUp!

WhatsUp es una herramienta para analizar tus chats de WhatsApp

Obtenga estadisticas y graficos sobre su actividad de mensajeria

Como utilizar WhatsUp?

  • Se exporta el chat que se quiere analizar
  • Se puede usar el archivo .zip exportado o el .txt descomprimido
  • Se arrastra el archivo al recuadro o se selecciona desde la pagina de WhatsUp
  • El sistema hace el resto y entrega el analisis de los datos

Que datos entrega WhatsUp?

  • Usuario que mas mensajes envio
  • Emojis mas utilizados
  • Franja horaria con mayor actividad
  • Dias con mayor cantidad de mensajes
  • Nube de palabras con las palabras mas frecuentes
